## Project Overview

Static Spotify clone using vanilla HTML, CSS, and JavaScript. No build tools or framework dependencies.

## Architecture

- `index.html` — Main entry point with all HTML structure
- `css/style.css` — Stylesheet with Spotify-style dark theme
- `js/script.js` — JavaScript with player controls and interactivity
- `assets/` — Image assets directory (currently uses inline SVG placeholders)

## Commands

No build, test, or lint commands available. Open `index.html` directly in a browser to view the application.

## Notes

- Images use inline SVG data URIs as placeholders — replace with actual assets in `assets/` if desired
- The player bar includes simulated playback progress when playing
